Git

  • cd into the folder that you want to create a new file in, then use "touch" and the file name to create that file
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • now you are working on the changes in the branch that you named
  • When you are done with the changes use git add -A to commit all files in the working branch
  • Then you need to actually commit the changes by using git commit -m "your message"
  • Then make sure you are on the main branch by typing git pull origin main
  • Finally push the changes by doing git push origin "branch name"
  • Then go to github and pull a new request, compare the main to the branch you are working on, create it, merge it
  • Finally you need to update your terminal so type git checkout main and then git pull origin main
